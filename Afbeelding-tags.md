HTML basis tags referentie
==========================

### Afbeelding tags

| Tag                  | Beschrijving                              |
| -------------------  | ----------------------------------------  |
| **`<img>`** | Definieert een afbeelding |
| **`<area>`** | Definieert een gebied binnen een imagemap |
| **`<map>`** | Definieert een afbeelding map |
